Objetivos:El alumno conocerá la configuración, la programación y el uso del módulo UART (Universal Asynchronous
Receiver Transmitter ) para implementar comunicaciones de tipo serial, particularmente bajo el est´andar RS-232.

Descripción: En esta prática se realizará la conexión  de las terminales RXD Y TXD de la tarjeta Raspberry con el adaptador UART  de la PC, también se realizará la configuración de la Raspberry para que se active este tipo de conexión. Para enviar y recibir datos en la Paspberry Pi es necesario usar un programa emulador. Se realizarán dos actividades donde se hará uso de esta comunicación tanto para enviar como para recibir datos.
